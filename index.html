Doom and Bloom

Custom 3D​ game engine and networked multiplayer game made in 10 weeks. I contributed all ​3D​ graphics and core gameplay code. Features Splatoon-like gameplay with collaborative texture painting over the network.

  • C++
  • OpenGL
  • GLSL shaders
  • Visual Studio
  • Blender

This was a team project class with no instruction. I'm proud of this game because it is fun to play and achieves a certain look. We did a good job of scoping our work for the time available so we were never caught without critical features implemented or with an unstable or unperformant build. Watch the video to see it in action! Features include batched instanced draws and culling, LOD flowers, mocked subsurface scatter, interactive foliage and wind vertex shaders, networked virtual texture painting, first/third person controls.

Doom and Bloom 2

Currently remaking the game in Unreal Engine 4 with the objective of learning industry tools and workflows.

  • C++
  • Unreal Engine 4
  • Visual Studio
  • Perforce

This is my current project. So far, we can paint over the network and the core gameplay mechanics of tilling, seeding, and watering are in place. Same as before, the server owns the ground truth virtual texture and multicasts minimally sized update regions of pixels to the clients.
